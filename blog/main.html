What is event bubbling in the DOM?

Event bubbling is a process in the DOM (Document Object Model) where an event
that happens on an element doesn’t just stay there—it "bubbles up" through its parent
elements, step by step, all the way to the top of the DOM. It’s called
"bubbling" because it moves upward, similar to bubbles rising in water.
